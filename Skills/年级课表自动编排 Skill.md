# 智能年级课表自动编排系统 · Skill 说明文档

> **Skill 名称**：`smart-schedule`  
> **安装路径**：`~/.workbuddy/skills/smart-schedule/`  
> **适用对象**：教务、年级组长、班主任  
> **核心目标**：零冲突 · 课时固定 · 教师固定 · 工作量均衡 · 全合规排课

---

## 一、快速上手

在对话框中说出以下任意表述，即可触发本 Skill：

```
帮我排一下高二年级的课表
自动排课，共14个班，文科6班，理科8班
生成年级课表
```

系统会提示你输入所需参数，之后自动完成全部排课工作。

---

## 二、输入参数说明

| 参数 | 示例 | 说明 |
|------|------|------|
| 年级名称 | 高二年级 | 用于课表标题显示 |
| 班级总数 | 14 | 文科+理科合计 |
| 文科班数 | 6 | 排政治/历史/地理 |
| 理科班数 | 8 | 排物理/化学/生物 |
| 各科教师名单 | 见下方示例 | JSON 或直接文字描述均可 |
| 班会偏好 | 上午/下午 | 对应第4节或第7节 |

## 三、强制固定规则（不可修改）

### 3.1 每日课时结构

```
早读 → 第1节 → 第2节 → 第3节 → 第4节 → 午休 → 第5节 → 第6节 → 第7节 → 晚自习1 → 晚自习2
```

### 3.2 每周课时硬标准

| 科目 | 节数/班/周 | 适用 |
|------|-----------|------|
| 语文 | 8节 | 全部班级 |
| 数学 | 8节 | 全部班级 |
| 英语 | 8节 | 全部班级 |
| 政治 | 6节 | 文科班 |
| 历史 | 6节 | 文科班 |
| 地理 | 6节 | 文科班 |
| 物理 | 6节 | 理科班 |
| 化学 | 6节 | 理科班 |
| 生物 | 6节 | 理科班 |
| 体育 | 2节 | 全部班级 |
| 信息 | 1节 | 全部班级 |
| 班会 | 1节 | 全部班级 |
| 自习 | 余量自动填充 | 正课时段每日≤1节 |
| 晚自习 | 每天2节(共10节) | 语数英轮换 |

### 3.3 班会硬规则

- ✅ 只能排在：**上午第4节** 或 **下午第7节**
- ✅ 每班每周仅 1 节
- ❌ 班会当天：不得有信息、不得有自习

### 3.4 自习硬规则

- ✅ 同一班级同一天正课时段 **最多1节自习**
- ❌ 自习当天：不得有信息、不得有班会

### 3.5 互斥规则（绝对禁止）

> **信息 · 自习 · 班会** 三者不得出现在同一天同一班

### 3.6 教师授课规则

| 规则 | 说明 |
|------|------|
| 一科一班一师 | 同一班级某科目，整周只能由同一位教师授课 |
| 一对多允许 | 一位教师可教多个班级 |
| 工作量均衡 | 同科目教师周课时差 **≤ 2节** |
| 时间冲突禁止 | 同一教师同一时段只能在一个班上课 |

## 四、排课优先级

```
Priority 1  ▶ 固定班会（上午第4节 / 下午第7节）
Priority 2  ▶ 班级-教师绑定（一科一师）
Priority 3  ▶ 语数英（各8节，均匀分散）
Priority 4  ▶ 副科（各6节，均匀不扎堆）
Priority 5  ▶ 体育（2节/周）
Priority 6  ▶ 信息（互斥日规避）
Priority 7  ▶ 自习（每日≤1节）
Priority 8  ▶ 晚自习填充（语数英轮换）
Priority 9  ▶ 全局冲突校验
```

## 六、命令行直接调用

```bash
# 安装路径
cd ~/.workbuddy/skills/smart-schedule

# 基本用法
python scripts/schedule.py \
  --grade "高二年级" \
  --classes 14 \
  --liberal 6 \
  --science 8 \
  --meeting-pref afternoon \
  --output 高二年级课表.md

# 带教师名单
python scripts/schedule.py \
  --grade "高二年级" \
  --classes 14 \
  --liberal 6 \
  --science 8 \
  --teachers teachers.json \
  --meeting-pref morning \
  --output 高二年级课表.md
```

## 七、禁止规则速查

| 规则 | 描述 |
|------|------|
| 主科单日上限 | 同一班同一天，语/数/英合计 ≤ 3节 |
| 主科连排限制 | 同班任何连续3节不得全为主科 |
| 体育/信息限制 | 不排早读、晚自习1、晚自习2 |
| 自习单日限制 | 同一班每天正课时段自习 ≤ 1节 |
| 三者互斥 | 信息/自习/班会不同日（同班） |
| 教师时间冲突 | 同一教师同一时段不得跨班 |
| 科目换师禁止 | 同一班一科一师，全周不变 |
